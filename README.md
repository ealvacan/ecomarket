# EcoMarket - Trueque de Productos Ecológicos

Plataforma para el intercambio de productos ecológicos y fomento de la sostenibilidad.

## Características

- CRUD completo de productos
- Generación de informes XML con visualización de árbol
- Cálculo de valores totales y porcentajes
- Diseño responsivo y moderno
- Desplegado en GitHub Pages

## Tecnologías

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript (ES6+)
- GitHub Pages

## URL del Proyecto

https://ealvacan.github.io/ecomarket

## Uso

1. Explora los productos ecológicos disponibles.
2. Agrega nuevos productos usando el botón "Agregar Producto".
3. Genera informes XML para ver el impacto ecológico.
4. Visualiza el árbol XML y los resúmenes de datos.
